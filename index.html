Acesse o repositório GitHub (demoiselle/behave)
dbehave - ferramenta open source para testes funcionais automatizados, que promove a aplicação de técnica de desenvolvimento Ágil baseada na abordagem Desenvolvimento Orientado por Comportamento / Behavior Driven Development – BDD. Esta abordagem visa encorajar a colaboração entre desenvolvedores, pessoas de negócio e setores de qualidade, na construção de projetos de software.
